Algemene vereisten
Jouw project moet minimaal aan de volgende voorwaarden voldoen:

Gebruik van minimum 3 gelinkte entiteiten (exclusief Identity-tabellen)
Moet logica bevatten (niet enkel CRUD-operaties)
Er moeten minimum 2 gebruikersrollen kunnen onderscheiden worden (bv.: Admin en Gebruiker)
Mag geen kopie zijn van een bestaande opdracht (geen kopie van Comme Chez Swa of webshop)
Technische vereisten
Je levert een .NET 8 MVC-applicatie op (geen API vereist!)
Je project dient de richtlijnen van Clean Architecture te volgen
Gebruik van Entity Framework Core en MS SQL Server is verplicht
Authenticatie en autorisatie aanwezig
Voldoende aandacht voor beveiliging (validatie, CSRF-aanvallen, etc.)
Je views bevatten een nette lay-out, niet enkel gegenereerde code
